Explanation: An Armstrong number means: take each digit, raise it to the power of the number of digits, and add them together.

153 (3 digits):

- First digit: 1³ = 1

- Second digit: 5³ = 125

- Third digit: 3³ = 27

Now add them: 1 + 125 + 27 = 153.

Since the sum is the same as the original number, 153 is an Armstrong number.
